Buatlah web service yang menyediakan layanan perhitungan hash dari Python Standard Library (https://docs.python.org/2/library/hashlib.html)
Fungsi-fungsi yang harus ada antara lain:

- md5
- sha1
- sha224

Library yang digunakan adalah Ladon untuk web service server (http://ladonize.org/ dan http://pythonhosted.org/ladon/) dan Suds untuk web service client (https://fedorahosted.org/suds/). 
Untuk Soal 1 ini, modifikasilah Example 1 pada Ladon example (http://pythonhosted.org/ladon/) dengan melengkapi fungsi-fungsi matematika yang ada pada pustaka hashlib.

Untuk bahasa pemrograman Java (J2EE) atau bahasa lain, bisa menggunakan library yang memenuhi requirement soal.
